# Hyperthermic Cancer Treatment Research Project

**Context:** Mathematical modelling study of hyperthermic cancer treatment using bioheat transfer equations.


**My Contribution:**
- Implemented a finite-difference (FTCS) solver for the Pennes bioheat equation
- Modelled volumetric SAR heat source and cooling boundary conditions
- Performed parameter sensitivity analysis on perfusion and heat input
- Generated temperature field visualisations using Matplotlib

**Output:**  
Produced quantitative heat distribution analysis and treatment depth estimates (~25mm penetration under enhanced modelling assumptions).
